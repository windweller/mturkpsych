Source code



Github Source Code

The source code of this project is packaged into a library called Tinker, developed and maintained by Allen Nie. The source code is written in Scala, and requires Scala runtime, and SBT to run. Tinker is still being actively developed at this point, and questions can be asked on github issue page: https://github.com/windweller/Tinker

Future Rules

All our future and past Tregex patterns can be found under CLIR (Computational Linguistics and Information Retrieval) github page: https://github.com/clir/time-perception.

Executable

We plan on releasing a jar executable for easy command-line integration, and a runnable enviornment (locally-run) in the following months.
